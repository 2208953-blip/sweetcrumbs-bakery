## Project Overview
This project is a functional bakery website front-end built strictly using **HTML Tables** for layout and **HTML attributes** for styling. The assignment focuses on understanding the core structure of HTML without the use of CSS or JavaScript.

## Features Included:
1.  **Product Gallery:** A 3x2 grid showcasing products with icons.
2.  **Testimonials:** Three customer reviews with star ratings.
3.  **Contact Form:** A complete form including text, email, tel, select, and textarea inputs.
4.  **Location Card:** Address and directions with a Google Maps link.
5.  **Multi-column Footer:** A 4-column footer containing site navigation and business info.

## Implementation Details
*   **Layout:** Managed through nested HTML tables.
*   **Styling:** Colors and spacing handled via `bgcolor`, `cellpadding`, and `font` tags.
*   **Validation:** Native HTML5 form validation for required fields.

## How to Run
Simply download the `index.html` file and open it in any web browser (Chrome, Edge, or Firefox).
